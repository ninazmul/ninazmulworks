A high-end, futuristic, and interactive personal portfolio engineered for technical authority. This platform showcases advanced proficiency in the MERN stack, Next.js, and architectural scalability, featuring a premium glassmorphic UI and production-grade software integration.

## 🚀 Technical Highlights

- **Framework**: Next.js 15+ (App Router) with Turbopack optimization.

- **Motion**: Advanced Framer Motion orchestrations for staggered entries and magnetic micro-interactions.
- **Data Integration**: Hybrid architecture merging static featured projects with dynamic, database-driven content via MongoDB & Mongoose.

## 🤸 Quick Start

**Prerequisites**

- [Node.js](https://nodejs.org/) (v18+)
- [npm](https://www.npmjs.com/) or [pnpm](https://pnpm.io/)
- [MongoDB](https://www.mongodb.com/) (for dynamic reviews and projects)

**Installation**

1. Clone the repository:
   ```bash
   git clone https://github.com/ninazmul/portfolio.git
   cd portfolio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables (.env.local):
   ```env
   MONGODB_URI=your_mongodb_uri
   EMAIL_USER=your_smtp_email
   EMAIL_PASS=your_smtp_password
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

## 🚀 Build & Deployment

To generate an optimized production bundle:

```bash
npm run build
```

Deployed via **Vercel** with automatic CI/CD integration.
